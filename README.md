# Lumina: Echo of Light Site

This is the dedicated web package for Lumina: Echo of Light.

It is separate from the personal portfolio site and contains only public-safe files for the game:

- `index.html` - a small public game information page
- `privacy-policy.html` - the Google Play privacy policy page
- `assets/` - public store art and promotional images
- `store-listing.md` - draft Google Play listing copy and publishing notes

Do not add Android signing keys, passwords, Play Console credentials, or private build artifacts to this repository.

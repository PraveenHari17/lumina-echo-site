# Lumina: Echo of Light

## Short Description

Explore shadowy ruins with echo pulses, dashes, puzzles, bosses, and daily challenges.

## Full Description

Lumina: Echo of Light is a glowing puzzle adventure where you guide a small light through ancient ruins, hidden paths, hazards, portals, enemies, and boss arenas.

Use Echo pulses to reveal the world around you, dash through danger, collect sparks and light cores, unlock upgrades, and return for daily challenges. Play solo or use local duo mode on one device with mobile joystick controls.
